Integrate Synaplan AI support chat widget into your WordPress site. Provide instant AI-powered customer support with knowledge base integration.

== Description ==

**Synaplan AI Support Chat** transforms your WordPress website with intelligent AI-powered customer support. This plugin provides a seamless wizard-based setup that creates a fully functional AI chat assistant in minutes.

= Key Features =

* **Easy Setup Wizard** - Step-by-step configuration process
* **AI-Powered Chat** - Intelligent responses using advanced AI models
* **Knowledge Base Integration** - Upload documents (PDF, DOCX) to enhance AI knowledge
* **Customizable Widget** - Adjust colors, position, and behavior to match your brand
* **Mobile Optimized** - Responsive design for all devices
* **Secure** - WordPress site verification and encrypted communication
* **No Email Confirmation** - Instant activation for WordPress users
* **File Search** - AI can search uploaded documents to answer questions

= How It Works =

1. **Install & Activate** the plugin
2. **Run the Setup Wizard** - Enter your email and create a password
3. **Configure Widget** - Choose colors, position, and welcome message
4. **Upload Documents** (Optional) - Add PDFs or DOCX files for AI to reference
5. **Done!** - Your AI chat widget is now live on your site

== Installation ==

= Automatic Installation =

1. Go to **Plugins** → **Add New** in your WordPress admin
2. Search for "Synaplan AI Support Chat"
3. Click **Install Now** and then **Activate**
4. Follow the setup wizard

= Manual Installation =

1. Download the plugin ZIP file
2. Go to **Plugins** → **Add New** → **Upload Plugin**
3. Choose the downloaded ZIP file and click **Install Now**
4. Click **Activate Plugin**
5. Follow the setup wizard

= After Activation =

1. Navigate to **Synaplan AI Support Chat** in your WordPress admin menu
2. Complete the 4-step setup wizard:
   * **Step 1:** Create your account (email & password)
   * **Step 2:** Configure widget appearance and behavior
   * **Step 3:** Upload knowledge base files (optional)
   * **Step 4:** Review and complete
3. Your chat widget is now active on all pages!

== Frequently Asked Questions ==

= Do I need a Synaplan account? =

The setup wizard automatically creates your Synaplan account during installation. No separate registration needed!

= Is email confirmation required? =

No! WordPress plugin users are instantly activated without email confirmation.

= How do I customize the AI responses? =

You can customize AI behavior at [app.synaplan.com](https://app.synaplan.com/index.php/prompts) using the Prompts Manager.

= Can I upload more documents later? =

Yes! Visit the [File Manager](https://app.synaplan.com/index.php/filemanager) to add more documents anytime.

= Where is my API key? =

Your API key is displayed in the WordPress admin dashboard. Click "Show API Key" to reveal it.

= Is my data secure? =

Yes! All data is encrypted and transmitted over HTTPS. Your conversations and documents are private and secure.

= What file formats are supported? =

PDF and DOCX files are supported for the knowledge base.

= Can I customize the widget appearance? =

Yes! You can customize colors, position, welcome message, and more from the Settings page.

= Does it work on mobile devices? =

Yes! The widget is fully responsive and optimized for all devices including smartphones and tablets.

= Can I use this on multiple sites? =

Yes! Each WordPress site gets its own configuration and can use the same or different Synaplan accounts.

This plugin connects to Synaplan's external API (app.synaplan.com) to provide AI chat functionality. When you use this plugin:

* User conversations are sent to Synaplan servers for AI processing
* Uploaded documents are stored and processed on Synaplan servers
* WordPress site verification tokens are exchanged during setup
* No personal data is collected without user consent
